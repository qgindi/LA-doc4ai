# Method `Au.csvTable.RemoveRow`

Removes one or more rows.

```
public void RemoveRow(int index, int count = 1)
```

##### Parameters

- *index*  (`int`):
    0-based row index.
- *count*  (`int`):
    How many rows to remove, default 1.

##### Exceptions

- `ArgumentOutOfRangeException`
- `ArgumentException`
