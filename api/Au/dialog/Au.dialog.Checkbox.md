# Method `Au.dialog.Checkbox`

Adds check box (if *text* is not `null`/empty).

```
public dialog Checkbox(string text, bool check = false)
```

##### Parameters

- *text*  (`string`)
- *check*  (`bool`)

##### Returns

`Au.dialog`

#### Remarks

To get check box state after closing the dialog, use `Au.dialog.Controls`.
