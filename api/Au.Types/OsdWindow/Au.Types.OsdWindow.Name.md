# Property `Au.Types.OsdWindow.Name`

OSD window name. Optional, default `null`.

```
public string Name { get; set; }
```

##### Property Value

`string`

#### Remarks

This text is invisible. Can be used to find OSD window. The class name is `"Au.OSD"`; if with shadow - `"Au.OSD2"`. This property cannot be changed after creating OSD window.
